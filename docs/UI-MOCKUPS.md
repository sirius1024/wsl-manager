底部状态栏显示 WSL 版本、内核版本、Windows 版本。

## 关键页面/弹窗

### 1. 新建实例

- 下拉选择发行版（来自 `wsl --list --online`）
- 输入自定义实例名（可选，留空使用默认名）
- “创建”按钮
- 提示：安装由 WSL 在后台执行，完成后请手动刷新

### 2. 删除确认

```
确认删除实例 "Ubuntu" 吗？
此操作不可恢复，实例内所有数据将被清除。

[取消]  [确认删除]
```

### 3. 重命名实例

- 输入新实例名
- 提示：重命名会导出并重新导入实例，原实例类型将变为 imported
- “确认重命名”按钮

### 4. 设置页面

- 默认终端选择（Windows Terminal / 系统默认）
- 自动刷新开关与间隔
- 主题（浅色/深色/跟随系统）
- 隐藏 Docker Desktop 内置实例

## 设计原则

- 简洁、信息密度适中
- 状态用颜色区分（运行=绿色，停止=灰色，错误=红色）
- 操作按钮按使用频率排序
- 所有危险操作使用二次确认
- 异步操作期间显示加载/禁用状态，避免重复提交
